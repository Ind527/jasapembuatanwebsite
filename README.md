## Getting Started

The website is already configured to run on Replit. The server automatically starts on port 5000.

### Local Development

If running locally:

```bash
node server.js
```

Then open http://localhost:5000 in your browser.

## Deployment

The website is configured for deployment using Replit's autoscale deployment. Simply click the "Deploy" button in Replit to publish your website.

## Customization

To customize the website:

1. **Colors**: Edit the Tailwind config in each HTML file's `<script>` section
2. **Content**: Modify text directly in the HTML files
3. **Images**: Replace image files in the root directory
4. **Contact Info**: Update WhatsApp numbers and links in the HTML files

## Cache Control

The server includes cache control headers to prevent browser caching during development, ensuring you always see the latest changes.
